
# Proyecto de Análisis Eólico con Windpowerlib

En esta carpeta se encuentra código desarrollado por mí para el análisis del recurso eólico en el departamento de La Guajira, Colombia, utilizando la librería Windpowerlib de Python.

Los datos meteorológicos empleados en este proyecto fueron obtenidos del Instituto de Hidrología, Meteorología y Estudios Ambientales (IDEAM). El objetivo principal es modelar y analizar el comportamiento de sistemas de generación eólica a partir de información climática real.

Este material ha sido desarrollado con fines académicos y de aprendizaje, como parte de mi formación en Ingeniería Eléctrica y mi interés en el sector de las energías renovables. Espero que pueda servir como referencia para estudiantes, investigadores y profesionales interesados en el modelado y evaluación de proyectos eólicos mediante herramientas de código abierto.

**Herramientas utilizadas**

* Python
* Windpowerlib
* Pandas
* NumPy
* Matplotlib

**Ubicación del caso de estudio**

* La Guajira, Colombia

**Fuente de datos**

* IDEAM (Instituto de Hidrología, Meteorología y Estudios Ambientales)
